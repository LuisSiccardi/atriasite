Atria Website
=============
Este es el sitio web finalizado para la asociación civil "Atria", dedicada a la Astronomía y Astronáutica.

Estructura de archivos:
-----------------------
atria-website/
├── index.html
├── about.html
├── education.html
├── research.html
├── tourism.html
├── activities.html
├── tools.html
├── contact.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── data/
│   └── activities.json
└── README.txt

Instrucciones:
--------------
1. Coloque todos los archivos y carpetas respetando la estructura anterior.
2. Para visualizar el sitio de forma dinámica (especialmente la carga del JSON en activities.html), es recomendable servirlo a través de un servidor web (por ejemplo, Apache, Nginx o usando un servidor de desarrollo).
3. Abra el archivo index.html en su navegador para comenzar a explorar el sitio.
